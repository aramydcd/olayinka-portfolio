Frontend: Bootstrap 5, JavaScript (Fetch API)

Data Tools: Power BI (DAX, Power Query), SQL

Environment: uv (Dependency Management)

------------------------------
⚙️ Installation & Setup
This project uses uv for extremely fast and reliable dependency management.

Clone the repository:

Bash
git clone https://github.com/aramydcd/olaynka-portfolio.git
cd olaynka-portfolio
Install dependencies:

Bash
uv sync
Setup Environment Variables:
Create a .env file based on .env.example:

Bash
cp .env.example .env
Run Migrations & Start Server:

Bash
uv run python manage.py migrate
uv run python manage.py runserver
